# 806. 写字符串需要的行数

- [🔗题目链接](https://leetcode-cn.com/problems/number-of-lines-to-write-string/)

简单模拟。

```kotlin
/**
 * @author Yananart
 */
class Solution {
    fun numberOfLines(widths: IntArray, s: String): IntArray {
        var lines = 1
        var last = 0
        for (ch in s.toCharArray()) {
            val now = widths[ch - 'a']
            if (now + last > 100) {
                lines++
                last = now
            } else {
                last += now
            }
        }
        return intArrayOf(lines, last)
    }
}
```
