# 172. 阶乘后的零

- [🔗题目链接](https://leetcode-cn.com/problems/factorial-trailing-zeroes/)

数学问题，统计有多少个5因数即可。

```kotlin
/**
 * @author Yananart
 */
class Solution {
    fun trailingZeroes(n: Int): Int {
        var now = 5
        var res = 0
        while (now <= n) {
            var t = now
            while (t % 5 == 0) {
                t /= 5
                res++
            }
            now++
        }
        return res
    }
}
```
