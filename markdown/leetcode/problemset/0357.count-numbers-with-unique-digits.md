# 357. 统计各位数字都不同的数字个数

- [🔗题目链接](https://leetcode-cn.com/problems/count-numbers-with-unique-digits/)

动态规划。

```kotlin
/**
 * @author Yananart
 */
class Solution {
    companion object {
        private val dp = IntArray(10)

        init {
            dp[1] = 10
            var now = 9
            for (i in 2..8) {
                now *= (11 - i)
                dp[i] = dp[i - 1] + now
            }
        }
    }

    fun countNumbersWithUniqueDigits(n: Int): Int {
        return dp[n]
    }
}
```
