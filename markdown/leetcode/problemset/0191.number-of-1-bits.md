# 191. 位1的个数

- [🔗题目链接](https://leetcode-cn.com/problems/number-of-1-bits/)

判断当前最右位是否为1(与1看是否等于1)，是的话结果++，然后右移一位，最多重复32次。

```kotlin
/**
 * @author Yananart
 */
class Solution {
    fun hammingWeight(n: Int): Int {
        var res = 0
        var now = n
        repeat(32) {
            if (now.and(1) == 1) {
                res++
            }
            now = now.shr(1)
        }
        return res
    }
}
```
