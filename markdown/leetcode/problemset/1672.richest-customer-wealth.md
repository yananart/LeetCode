# 1672. 最富有客户的资产总量

- [🔗题目链接](https://leetcode-cn.com/problems/richest-customer-wealth/)

简单模拟。

```kotlin
/**
 * @author Yananart
 */
class Solution {
    fun maximumWealth(accounts: Array<IntArray>) =
    IntArray(accounts.size) {
        var sum = 0
        for (item in accounts[it]) {
            sum += item
        }
        sum
    }.let {
        var max = 0
        for (item in it) {
            max = kotlin.math.max(item, max)
        }
        max
    }
}
```
