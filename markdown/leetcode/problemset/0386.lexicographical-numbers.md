# 386. 字典序排数

- [🔗题目链接](https://leetcode-cn.com/problems/lexicographical-numbers/)

DFS。

```kotlin
/**
 * @author Yananart
 */
class Solution {
    fun lexicalOrder(n: Int): List<Int> {
        val res = ArrayList<Int>(n)
        for (i in 1..9) {
            dfs(i, n, res)
        }
        return res
    }

    private fun dfs(now: Int, max: Int, res: ArrayList<Int>) {
        if (now <= max) {
            res.add(now)
            if (now < max) {
                for (i in 0..9) {
                    dfs(10 * now + i, max, res)
                }
            }
        }
    }
}
```
