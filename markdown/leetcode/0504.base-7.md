# 504. 七进制数

按规则转化即可，按给出的数据范围，最高处理到7^9即可，注意负数转为正数处理。

```kotlin
import kotlin.math.abs

/**
 * @author Yananart
 */
class Solution {
    fun convertToBase7(num: Int): String {
        // 最多到7的9次方
        val neg = num < 0
        var now = abs(num)
        var ans = ""
        for (i in 1..9) {
            ans += ((now % 7).toString())
            val next = now / 7
            if (next == 0) break
            now = next
        }
        ans = ans.reversed()
        return if (neg) "-$ans" else ans
    }
}
```
